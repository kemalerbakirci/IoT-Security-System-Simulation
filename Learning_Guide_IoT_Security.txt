## 2Ô∏è‚É£ What is MQTT?
- A lightweight messaging protocol for IoT.
- Uses **Publish/Subscribe model**.
- Example: Door sensor publishes data, Node-RED subscribes to display it.

## 3Ô∏è‚É£ Key MQTT Concepts
- **Topics:** Channels where messages are published (e.g., home/security/door).
- **QoS:** Message delivery guarantee levels (0, 1, 2).
- **Persistent Sessions:** Broker remembers subscriptions if `clean_session=False`.
